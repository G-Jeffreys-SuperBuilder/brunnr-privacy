Privacy Policy

Last updated: October 2025

Overview

The Brunnr Educational Enhancement Extension ("the Extension") provides reading comprehension enhancement features for MathAcademy lessons. This policy explains what information we collect, how it is used, and the rights of users and schools.

Information We Collect

  • User identification: Chrome profile email address obtained via Chrome Identity API for access validation and progress tracking.
  • Lesson context: Current lesson ID, lesson number, topic ID, and page URL from MathAcademy.
  • Reading timer data: Timer start and completion events; reading time duration calculated based on explanation length; which timers have been completed.
  • Learning performance data: Question attempt results (correct/incorrect), time spent on lessons, completion status, which specific questions were answered.
  • Session information: Unique session ID generated per browser session, timestamps for all interactions.
  • Timer completion history: Which explanation reading timers have been completed (stored locally to prevent re-gating).

How We Use Information

  • To verify school membership and provide access to enhanced learning features.
  • To calculate appropriate reading time based on explanation length (100 words per minute).
  • To enforce reading time requirements when students answer questions incorrectly.
  • To track learning progress and feature effectiveness for educational improvement.
  • To prevent re-gating of previously completed reading timers across sessions.
  • To help teachers and administrators understand student engagement and performance.

Data Storage and Transmission

Local Storage: Timer completion history and session data are stored locally in Chrome storage to maintain state across page loads and sessions.

Remote Storage: Analytics and performance data are transmitted securely over HTTPS to our AWS-hosted backend infrastructure.

Data Sharing and Retention

We do not sell, rent, or share personal data with any third parties outside of the educational institution. Data is used exclusively for educational purposes within approved school programs. Analytics data is retained for the duration of the school year for reporting purposes, after which it may be anonymized for aggregate analysis.

Student Data and COPPA / FERPA Compliance

This extension is designed for use in K-12 educational settings where the school provides appropriate consent on behalf of students under the Children's Online Privacy Protection Act (COPPA). All student data is processed in accordance with the Family Educational Rights and Privacy Act (FERPA). Student data remains under the control of the educational institution and is used solely for educational purposes.

Reading-Time Gating Mechanism

When a student answers a question incorrectly, the Extension temporarily disables the "Continue" button and displays a countdown timer. The timer duration is calculated based on the explanation's word count at a reading rate of 100 words per minute (minimum 10 seconds). This ensures students take adequate time to read and understand explanations before progressing. The system remembers completed timers locally to avoid unnecessary repetition on the same question in future sessions.

Browser Permissions

The Extension requests the minimal permissions necessary to operate:

  • identity & identity.email: To verify school membership via Chrome profile email
  • storage: To save timer completion history and session data locally
  • activeTab: To detect MathAcademy lessons and implement reading gates
  • https://*.mathacademy.com/*: To modify lesson pages and control Continue buttons during reading periods
  • AWS API Gateway domains: To send analytics and validate user access

No browsing data from other websites is collected.

Security

All communications are encrypted via TLS 1.2 or higher. Data at rest is encrypted using AWS KMS-managed keys. Access to stored data is restricted to authorized school personnel and system administrators. The Extension includes retry logic for failed analytics to ensure no data loss while maintaining user experience.

User Rights and Controls

  • Clear Session Data: Users can clear all session data through the Extension popup interface.
  • Timer Persistence: Once a reading timer is completed for a specific question, it won't re-gate that question in future sessions.
  • Data Deletion: Schools or parents may request deletion of student data by contacting us.
  • Access Removal: Schools can remove user access at any time by updating the approved users list.

Data Minimization

The Extension only activates on MathAcademy domains. Data is only collected during active lesson sessions. Non-approved users experience no modifications to their MathAcademy experience, and no data is collected from them beyond the initial access verification check.

Children's Privacy

This Extension is intended for use in educational settings with appropriate school consent. We do not knowingly collect personal information from children under 13 without school authorization. If we learn that we have collected personal information from a child without proper consent, we will delete that information promptly.
